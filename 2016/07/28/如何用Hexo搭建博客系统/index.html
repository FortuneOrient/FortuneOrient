如何用 Hexo 3.0 搭建自己的博客系统

发表于 2016-07-28

本文介绍如何在Linux环境下，使用Hexo 3.0搭建简单的博客系统

Hexo 简单介绍

Hexo官方的描述是:

A fast, simple & powerful blog framework

官方网址:https://hexo.io/。网站支持简体中文，赞。
Hexo支持Markdown解析，有多个主题可供选择，能将本地博客同步至Github（或其他托管服务）。

Hexo下载与部署

前提

请确保本地环境安装了Node.js和Git。

安装

使用 npm 安装 Hexo。

$ npm install -g hexo-cli

建站

在本地新建一个文件夹，比如myblog，依次执行如下命令：

$ hexo init myblog
$ cd myblog
$ npm install

网站配置信息存放在文件_config.yml中，可参考官方说明，进行修改。

服务器

安装 hexo-server 模块

$ npm install hexo-server --save

同步

将本地博客同步到Github上，请参考官方文档 - 部署
