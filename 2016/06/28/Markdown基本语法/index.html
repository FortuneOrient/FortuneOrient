Markdown基本语法

发表于 2016-06-28

Markdown介绍，基本语法，注意事项

转义符号

在需要转义的符号之前，加上\。参考

标题设置

主次标题

  1. 通过在标题文字下方增加若干“=”，设置主要标题（一级标题）。
  2. 通过在标题文字下方增加若干“-”，设置次要标题（二级标题）。

多级标题

通过在标题文字前方增加若干“#”，设置标题等级。一共有1至6级。

文字强调

粗体

通过在文字前后各增加2个“*”或”_“，可设置为粗体。
举例：粗体

斜体

通过在文字前后各增加1个“*”或”_“，可设置为斜体。
举例：斜体

删划

通过在文字前后各增加2个“~“，可设置为删划字体。
举例： 删除

列表

无序列表

通过在文字前方增加1个“*”、”+”或“-”，再加上一个空格，可设置无序列表。
特别注意，“*”、”+”或“-”与文字之间，必须有一个空格。并且建议，同一文档中，只采用一种符号标明无须列表。

有序列表

通过在文字前方增加1个“阿拉伯数字”，再加上一个空格，可设置有序列表。
特别注意，“阿拉伯数字”与文字之间，必须有一个空格。列表的序号和排位，与数字的数值、大小、重复与否，都没有关系。

链接

Automatic Links

URLs:

<http://example.com/>

Email: 会被自动转义

<address@example.com>

内联方式

This is an [example link](http://example.com/ "With a Title").

引用方式

I get 10 times more traffic from [Google][1] than from [Yahoo][2] or [MSN][3].  

[1]: http://google.com/        "Google" 
[2]: http://search.yahoo.com/  "Yahoo Search" 
[3]: http://search.msn.com/    "MSN Search"

图片

内联方式

![alt text](/path/to/img.jpg "Title")

引用方式

![alt text][id]

[id]: /path/to/img.jpg  "Title"

代码

行内代码

通过在代码前后各增加1个“`“，可设置为行内代码。
举例：
这是文字I am code又是文字

大块代码

通过在代码块前使用Tab或四个空格。
举例：

Code line 1
Code line 2
Code line 3
...

引用

通过在文字前方增加1个“>”，再加上一个空格，可设置引用格式。
举例：

BlockQuote line 1
BlockQuote line 2
BlockQuote line 3
…

分割线

在空白行下方添加3个“-”或3个“*”。
