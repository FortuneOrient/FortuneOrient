Linux下的压缩命令详解

发表于 2016-06-12

Linux下的压缩程序：tar，gzip、gunzip，bzip2、bunzip2，compress、uncompress，zip、unzip，rar、unrar。
Linux下的压缩文件：.tar、.gz、.tar.gz、.tgz、.bz2、.tar.bz2、.Z、.tar.Z、.zip、.rar。

tar

Linux下常用的打包程序是tar，打出的包称为tar包，通常以.tar结尾。首先讲讲tar命令的基本用法：

tar命令选项很多，可以用man tar查看。

  1. 第一组，五个独立的命令，压缩或解压至少要用到其中一个。他们可以和其他组的命令连用，但不能和本组其他命令连用。
    -c: 建立压缩档案
    -x：解压
    -t：查看内容
    -r：向压缩归档文件末尾追加文件
    -u：更新原压缩包中的文件

  2. 第二组，这些参数是根据需要，在压缩或解压时选用的。
    -z：有gzip属性的
    -j：有bz2属性的
    -Z：有compress属性的
    -v：显示所有过程
    -O：将文件解开到标准输出

  3. 第三组，下面这个参数-f是必须的
    -f: 使用档案名字。这个参数是最后一个参数，后面只能接档案名。

下面举例说明：
# tar -cf all.tar *.jpg
将所有.jpg的文件打成一个名为all.tar的包。-c是表示产生新的包，-f指定包的文件名。
# tar -rf all.tar *.gif
将所有.gif的文件增加到all.tar的包里面去。-r是表示增加文件。
# tar -uf all.tar logo.gif
更新原来tar包all.tar中logo.gif文件，-u是表示更新文件。
# tar -tf all.tar
列出all.tar包中所有文件，-t是表示列出文件。
# tar -xf all.tar
解出all.tar包中所有文件，-x是解开的意思。

调用

以上是tar的基本用法。更进一步，tar可以在打包或解包的同时调用其它的压缩程序，比如调用gzip、bzip2等。

  1. tar调用gzip
    gzip是GNU组织开发的压缩程序，其压缩文件以.gz结尾。与gzip相对的解压程序是gunzip。tar命令使用-z参数调用gzip。举例说明：
    # tar -czf all.tar.gz *.jpg
    是将所有.jpg的文件打成一个tar包，并且将其用gzip压缩，生成一个gzip压缩过的包，包名为all.tar.gz
    # tar -xzf all.tar.gz
    是将上面产生的包解开。
  2. tar调用bzip2
    bzip2是压缩能力更强的压缩程序，其压缩文件以.bz2结尾。与bzip2相对的解压程序是bunzip2。tar命令使用-j参数调用bzip2。举例说明：
    # tar -cjf all.tar.bz2 *.jpg
    是将所有.jpg的文件打成一个tar包，并且将其用bzip2压缩，生成一个bzip2压缩过的包，包名为all.tar.bz2
    # tar -xjf all.tar.bz2
    是将上面产生的包解开。
  3. tar调用compress
    compress是另一个压缩程序，其压缩文件以.Z结尾。与compress相对的解压程序是uncompress。tar命令使用-Z参数调用compress。举例说明：
    # tar -cZf all.tar.Z *.jpg
    是将所有.jpg的文件打成一个tar包，并且将其用compress压缩，生成一个compress压缩过的包，包名为all.tar.Z
    # tar -xZf all.tar.Z
    是将上面产生的包解开。

小结

下面对tar系列的压缩文件作一个小结：

  1. 对于.tar结尾的文件
    # tar -xf all.tar
  2. 对于.gz结尾的文件
    # gzip -d all.gz 或 # gunzip all.gz
  3. 对于.tar.gz(或.tgz)结尾的文件
    # tar -xzf all.tar.gz (或# tar -xzf all.tgz)
  4. 对于.bz2结尾的文件
    # bzip2 -d all.bz2 或 # bunzip2 all.bz2
  5. 对于tar.bz2结尾的文件
    # tar -xjf all.tar.bz2
  6. 对于.Z结尾的文件
    # uncompress all.Z
  7. 对于.tar.Z结尾的文件
    # tar -xZf all.tar.Z

Windows文件

另外对于Window下的常见压缩文件.zip和.rar，Linux也有相应的方法来解压它们：

  1. 对于.zip
    Linux下提供了免费的程序：zip程序和unzip程序。zip是压缩程序，unzip是解压程序。举例说明：
    # zip all.zip *.jpg
    是将所有.jpg的文件压缩成一个zip包。
    # unzip all.zip
    是将all.zip中的所有文件解压出来。
  2. 对于.rar
    Linux下没有免费程序。处理.rar文件，需要安装RAR for Linux。安装后就有了rar和unrar这两个程序，rar是压缩程序，unrar是解压程序。举例说明：
    # rar a all *.jpg
    是将所有.jpg的文件压缩成一个rar包，名为all.rar，该程序会将.rar扩展名将自动附加到包名后。
    # unrar e all.rar
    是将all.rar中的所有文件解压出来。
